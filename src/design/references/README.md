# UI References

## Primary reference – Desktop

- petplatform-desktop.png
  - Hero split en dos columnas
  - Adoption izquierda (verde pastel, emocional)
  - Lost & Found derecha (coral pastel, urgencia suave)
  - CTA claros y separados
  - Search central con toggle Adopt / Lost&Found
  - Stats de impacto por tipo (adoption / reunion)

## Mobile reference

- petbridge-mobile.png
  - Hero split colapsa en cards verticales
  - Adoption card primero, Lost & Found debajo
  - CTA grandes, accesibles con una mano
  - Toggle Adopt / Lost&Found persiste
  - Diseño mobile-first, limpio y amigable

## UX notes

- Desktop y mobile comparten el mismo modelo mental
- El usuario siempre entiende:
  - qué tipo de mascota está viendo
  - qué acción puede hacer
- Lost & Found debe sentirse un poco más urgente que Adoption
